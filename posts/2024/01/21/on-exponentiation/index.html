Computation

Now back to the beginning.

Well, that's actually another question.

The power series definition already gives a hint of how exponentiation can be computed to arbitrary precision using elementary operations the computer understands (and using maybe Newton's method for the inversion), and the matrix diagonalization shows the kind of specific optimization we can employ to handle a specific class of problems. These are deep and vast topics I don't plan to cover here.

Even the simplest case is not as simple as it seems. Say we want to calculate . This naive python algorithm would take 64 steps:

def exp(a, b):
    r = 1
    for _ in range(b):
        r *= a
    return r

However, using this "exponentiation by squaring" alternative, we can reduce it to 8 steps:

# assuming a and b are integers and b is nonnegative 
def exp2(a, b):
    r = 1
    while b:
        if b & 1:
            r *= a
        a = a * a
        b >>= 1
    return r
